Visual Distinction Checklist

  • ✓ Each item has unique gradient background
  • ✓ Each item has distinct icon/symbol
  • ✓ Border thickness indicates item quality/rarity
  • ✓ Glow effects draw attention to collectible items
  • ✓ Premium items (Holy Sword, Super Potion) have enhanced effects
  • ✓ All items have glow + float animations
  • ✓ Color coding matches item type (potions=pink, gems=colors, etc.)
  • ✓ Equipment progression is visually clear (Iron → Steel → Holy)
