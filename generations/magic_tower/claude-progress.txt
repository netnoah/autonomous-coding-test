FEATURE IMPLEMENTED
================================================================================

✅ STAIRS ANIMATED INDICATOR (Test #1967)

Implemented animated glow indicators for stairs up and stairs down to help
players identify navigation points in the tower.

**Implementation Details:**

1. CSS Animations (src/index.css):
   - Added @keyframes stairsUpGlow with 2-second cycle
     * Upward-pointing glow effect (box-shadow moves upward)
     * Brightness pulsing from 100% to 120%
     * Vertical translation: 0px to -2px
     * Purple glow color (rgba(168, 85, 247))

   - Added @keyframes stairsDownGlow with 2-second cycle
     * Downward-pointing glow effect (box-shadow moves downward)
     * Brightness pulsing from 100% to 120%
     * Vertical translation: 0px to 2px
     * Darker purple color (rgba(147, 51, 234))

2. GameMap Component (src/components/GameMap.jsx):
   - Added isStairsTile() function to detect stairs tiles
   - Applied stairs-up-animation class to STAIRS_UP tiles
   - Applied stairs-down-animation class to STAIRS_DOWN tiles
   - Animations trigger automatically on tile render

3. Visual Differentiation:
   - Stairs up: Upward glow with -2px vertical movement
   - Stairs down: Downward glow with +2px vertical movement
   - Different purple shades for visual distinction
   - Both animations loop smoothly with 2-second duration

**Files Modified:**
- src/index.css: Added stairsUpGlow and stairsDownGlow animations and classes
- src/components/GameMap.jsx: Added isStairsTile() function and animation classes
- feature_list.json: Marked test #1967 as passing

**Technical Implementation:**
```css
/* Stairs Up Animation */
@keyframes stairsUpGlow {
  0%, 100% {
    box-shadow: 0 0 5px rgba(168, 85, 247, 0.4),
                0 -2px 10px rgba(168, 85, 247, 0.3),
                0 -4px 15px rgba(168, 85, 247, 0.2);
    transform: translateY(0);
    filter: brightness(1);
  }
  50% {
    box-shadow: 0 0 15px rgba(168, 85, 247, 0.7),
                0 -2px 20px rgba(168, 85, 247, 0.5),
                0 -4px 30px rgba(168, 85, 247, 0.3);
    transform: translateY(-2px);
    filter: brightness(1.2);
  }
}

/* Stairs Down Animation */
@keyframes stairsDownGlow {
  0%, 100% {
    box-shadow: 0 0 5px rgba(147, 51, 234, 0.4),
                0 2px 10px rgba(147, 51, 234, 0.3),
                0 4px 15px rgba(147, 51, 234, 0.2);
    transform: translateY(0);
    filter: brightness(1);
  }
  50% {
    box-shadow: 0 0 15px rgba(147, 51, 234, 0.7),
                0 2px 20px rgba(147, 51, 234, 0.5),
                0 4px 30px rgba(147, 51, 234, 0.3);
    transform: translateY(2px);
    filter: brightness(1.2);
  }
}
```

**User Experience:**
- Players can now easily identify stairs on each floor
- Animated glow draws attention to navigation points
- Different visual indicators for up vs down stairs
- Smooth looping animation doesn't distract from gameplay
- Helps players quickly find their way through the tower
