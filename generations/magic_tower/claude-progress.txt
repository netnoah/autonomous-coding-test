IMPLEMENTATION DETAILS
================================================================================

**CSS Structure:**
Each item has its own CSS class with:
- Radial gradient background for depth and visual appeal
- Unique border (2px for common, 3px for better items)
- Box shadow effects (stronger for higher quality items)
- ::before pseudo-element with unique emoji/symbol icon
- itemGlow animation (pulsing glow effect)
- itemFloat animation (subtle floating motion)

**Premium Items:**
Holy Sword, Holy Shield, and Super Potion have enhanced effects:
- Stronger glow animations (faster pulse)
- Brighter, more prominent colors
- Multiple layered box-shadows
- Thicker borders (3px instead of 2px)

**Example Item Styling:**
```css
/* Small Potion - Common healing item */
.tile-small-potion {
  background: radial-gradient(circle at 30% 30%, #FFB6C1 0%, #FF91A4 50%, #FF6C87 100%);
  border: 2px solid #E55A6F;
  box-shadow: inset 0 0 10px rgba(255, 255, 255, 0.6),
              0 0 10px rgba(255, 150, 170, 0.4);
}
.tile-small-potion::before {
  content: '⚗️';
  font-size: 24px;
  text-shadow: 0 2px 3px rgba(0, 0, 0, 0.4);
}

/* Holy Sword - Premium weapon */
.tile-holy-sword {
  background: radial-gradient(circle at 30% 30%, #FFFFE0 0%, #FFFACD 30%, #FFEC8B 60%, #FFD700 100%);
  border: 3px solid #CC9900;
  box-shadow: inset 0 0 15px rgba(255, 255, 255, 0.9),
              0 0 15px rgba(255, 255, 150, 0.7),
              0 0 20px rgba(255, 215, 0, 0.5);
  animation: itemGlow 1s ease-in-out infinite, itemFloat 2s ease-in-out infinite;
}
.tile-holy-sword::before {
  content: '⚔️';
  font-size: 26px;
  text-shadow: 0 2px 4px rgba(0, 0, 0, 0.5);
}
```

**Item Animations:**
All items have two animations:
1. itemGlow (2s): Pulsing glow effect that draws attention
2. itemFloat (3s): Subtle up-and-down floating motion

Premium items have faster glow animation (1s or 1.5s) for more prominence.

**GameMap Component Changes:**
- Updated getTileColor() to return item-specific CSS classes instead of Tailwind classes
- Updated getTileSymbol() to return empty string for items (icons now handled by CSS)
- All item tiles automatically get glow and float animations via CSS selectors

**Visual Hierarchy:**
- Common items: 2px border, standard animations
- Better items: 3px border, enhanced effects
- Premium items: Special golden glow, faster animations, stronger shadows
