Before:

After:
```jsx
<span className="text-gray-300">⚔️ ATK</span>
<span className="text-gray-300">🛡️ DEF</span>
```

**Rationale:**
Emoji variation selectors (️) ensure consistent rendering across different
platforms and browsers. Without them, some platforms display different styles
or colored versions of the same emoji. By standardizing on variation selector
emojis, we guarantee a uniform appearance throughout the UI.

**Benefits:**
1. Visual consistency - all icons have the same style
2. Cross-platform compatibility - uniform appearance on Windows, Mac, Linux, mobile
3. Better accessibility - consistent icon recognition
4. Professional appearance - polished, intentional design

- StatusPanel.jsx: Stats display (ATK, DEF)
