IMPLEMENTATION DETAILS
================================================================================

**CSS Structure:**
Each monster has its own CSS class with:
- Radial gradient background for depth
- Unique border (2px for easy/medium, 3px for hard, 4-6px for bosses)
- Box shadow effects (stronger for more powerful monsters)
- ::before pseudo-element with unique emoji/icon
- monster-idle animation class applied

**Example Monster Styling:**
```css
/* Green Slime - Easy monster */
.tile-green-slime {
  background: radial-gradient(circle at 30% 30%, #66CC66 0%, #44AA44 50%, #338833 100%);
  border: 2px solid #227722;
  box-shadow: inset 0 0 6px rgba(255, 255, 255, 0.3),
              0 2px 4px rgba(0, 0, 0, 0.2);
}
.tile-green-slime::before {
  content: '●';
  font-size: 28px;
  color: rgba(255, 255, 255, 0.6);
  text-shadow: 0 1px 2px rgba(0, 0, 0, 0.3);
}

/* Tower Lord - Final Boss */
.tile-tower-lord {
  background: radial-gradient(circle at 30% 30%, #FF0044 0%, #DD0033 50%, #BB0022 100%);
  border: 6px solid #990011;
  box-shadow: inset 0 0 20px rgba(255, 100, 150, 0.8),
              0 0 20px rgba(220, 0, 50, 0.6),
              0 0 30px rgba(180, 0, 30, 0.4),
              0 0 40px rgba(150, 0, 20, 0.2);
  animation: towerLordPulse 3s ease-in-out infinite;
}
.tile-tower-lord::before {
  content: '👿';
  font-size: 34px;
  text-shadow: 0 3px 6px rgba(0, 0, 0, 0.8);
}
```
