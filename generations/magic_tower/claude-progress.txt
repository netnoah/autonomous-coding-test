   - Screenshots captured showing game with animated monsters

**Files Modified:**
- src/index.css: Added monsterIdle @keyframes and .monster-idle class
- src/components/GameMap.jsx: Added isMonsterTile() function and applied CSS class

**Technical Implementation:**
```css
@keyframes monsterIdle {
  0%, 100% {
    transform: translateY(0) scale(1);
    opacity: 1;
  }
  25% {
    transform: translateY(-2px) scale(1.02);
    opacity: 0.95;
  }
  50% {
    transform: translateY(-3px) scale(1.03);
    opacity: 0.9;
  }
  75% {
    transform: translateY(-2px) scale(1.02);
    opacity: 0.95;
  }
}

.monster-idle {
  animation: monsterIdle 2s ease-in-out infinite;
}
```

**Visual Effect:**
- Monsters gently bob up and down every 2 seconds
- Subtle scaling adds "breathing" effect
- Smooth, professional animation that doesn't distract from gameplay
- All 15 monster types share the same idle animation for consistency
